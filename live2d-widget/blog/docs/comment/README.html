评论

::: tip

Aurora主题加入了评论，使用Waline | Waline集成评论，Valine是一款基于 Valine 衍生的简洁、安全的评论系统。

:::

获取serverURL

请移步到Waline官网，根据他们的教程，获取serverURL

快速上手 | Waline

配置

在配置文件config.js文件中，

module.exports = {

    themeConfig: {
        comment: {
            //是否显示评论
            showComment: true,
            serverURL: '上一步得到的serverURL地址',
        },
    }
}

关于评论的详细配置项，请看快速上手 | Waline，目前主题集成上述项配置，如需Waline其他的配置项，请自行克隆主题源码进行更改，评论组件位置docs/.vuepress/theme/lib/client/components/global/Comment.vue,所有的配置都是在docs/readme.md中进行配置

所有评论配置项，必须放在comment对象中

组件使用

Comment是一个全局组件，目前默认只在关于页面，文章页面以及友情链接页面进行使用，如果在其他页面使用，可以直接<Comment/>
